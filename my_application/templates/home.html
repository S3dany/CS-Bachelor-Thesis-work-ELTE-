Welcome to Analysaction


Analysaction is an algorithm that analyzes foreign transaction networks.


the tool promotes an alternative to traditional money transfers where foreign transactions can be replaced with local ones which means less fees.

How to use Analysaction?

  • Create an account.

  • Upload your csv file containing records of transactions between different countries.

  • The algorithm will run over your file and produces a report of your analysed data.

  • preview your saved reports that shows the amounts and transactions which could be replaced with local ones.

  • for more details about your report, you can download output.xlsx file which contains every aspect of the analysis.
